This talk, delivered by Online Reference was held on Wednesday October 12, 2016 in Online.

Abstract

<div class="markdown"><p><strong>Note</strong>: This talk is not currently officially scheduled. We would like to cover it at some point before Akshay’s talk following.</p> <p>We will cover some ideas from number theory like quadratic reciprocity, Euler’s totient function, and the Legendre and Jacobi symbols. We will use <a href="http://golibgen.io/view.php?id&#61;733865">Baker, 1984</a> as a reference.</p> </div> <div class="markdown"><h2>Quadratic Reciprocity</h2> <p>Let &#36;p&#36; be a prime number, and consider the additive group</p> &#36;\mathbf&#123;Z&#125;/p\mathbf&#123;Z&#125;&#36; <p>which is a finite group of order &#36;p&#36;. This group contains elements of the form</p> &#36;\&#123;\mathbf&#123;Z&#125;, \mathbf&#123;Z&#125;&#43;1, \dots, \mathbf&#123;Z&#125; &#43; &#40;p-1&#41; \&#125;&#36; <p>which is naturally isomorphic to the set</p> &#36;\&#123;0, 1, \dots, p-1\&#125;&#36; <p>For those who have taken MATH 145, note that the former set is the set of equivalence classes of the integers, modulo &#36;p&#36;, and the latter set is the set of canonical representatives for those equivalence classes.</p> <p>The additive group &#36;\mathbf&#123;Z&#125;/p\mathbf&#123;Z&#125;&#36; has additional structure. It inherits an associative and commutative multiplication operation from the integers; this multiplication operation remains well-defined on this quotient group. Furthermore, each element admits a multiplicative inverse. Thus, &#36;\mathbf&#123;Z&#125;/p\mathbf&#123;Z&#125;&#36; is actually a field.</p> <h2>Quadratic Residues</h2> <p>Consider the sequence of perfect squares, modulo prime number &#36;p&#36;. As an example, set &#36;p&#61;11&#36;. Then the perfect squares are:</p> &#36;\begin&#123;align&#125; 0^2 &amp;&#61; 0 \ &#123;&#40;-1&#41;&#125;^2 &#61; 1^2 &amp;&#61; 1 \ &#123;&#40;-2&#41;&#125;^2 &#61; 2^2 &amp;&#61; 4 \ &#123;&#40;-3&#41;&#125;^2 &#61; 3^2 &amp;&#61; 9 \ &#123;&#40;-4&#41;&#125;^2 &#61; 4^2 &amp;&#61; 5 \ &#123;&#40;-5&#41;&#125;^2 &#61; 5^2 &amp;&#61; 3 \end&#123;align&#125;&#36; <p>Since these are all the integers modulo &#36;11&#36;, the only possible perfect squares, modulo &#36;11&#36;, are &#36;\&#123;0, 1, 3, 4, 5, 9\&#125;&#36;. These numbers are known as the <strong>quadratic residues</strong> modulo &#36;11&#36;.</p> <p>In general, given prime number &#36;p&#36;, the quadratic residues modulo &#36;p&#36; are the elements &#36;n\in\mathbf&#123;Z&#125;/p\mathbf&#123;Z&#125;&#36; such that there exists &#36;q\in\mathbf&#123;Z&#125;/p\mathbf&#123;Z&#125;&#36; with &#36;q^2&#61;n&#36;. We can extend this concept to the integers by considering the canonical projection of an integer &#36;z&#36; onto quotient group &#36;\mathbf&#123;Z&#125;/p\mathbf&#123;Z&#125;&#36;. That is, an integer &#36;z&#36; is a quadratic residue modulo &#36;p&#36; if its canonical projection modulo &#36;p&#36; is a quadratic residue modulo &#36;p&#36;.</p> <p>Modulo an odd prime number &#40;like &#36;p&#61;11&#36;&#41;, there will be exactly &#36;\frac&#123;p-1&#125;&#123;2&#125;&#36; non-residues, and exactly &#36;\frac&#123;p-1&#125;&#123;2&#125;&#36; non-zero residues. Again, I state this without proof, but the proof is not too difficult.</p> <h2>Legendre Symbol</h2> <p>Let &#36;p&#36; be an odd prime number, and let &#36;a&#36; be an integer. Define the Legendre Symbol &#36;\left&#40;\frac&#123;a&#125;&#123;p&#125;\right&#41;&#36; as follows:</p> &#36;\left&#40;\frac&#123;a&#125;&#123;p&#125;\right&#41; &#61; \begin&#123;cases&#125; 0 &amp; \text&#123;if &#125;p\mid a \ -1 &amp; \text&#123;if &#125;a\text&#123; is a quadratic nonresidue modulo &#125;p \ 1 &amp;\text&#123;if &#125;a\text&#123; is a quadratic residue modulo &#125;p\text&#123;, and &#125;p\not\mid a \end&#123;cases&#125;&#36; <h2>Euler’s Criterion</h2> <p>Let &#36;p&#36; be an odd prime number, and let &#36;a&#36; be an integer. Then &#40;everything being modulo &#36;p&#36;&#41;</p> &#36;\left&#40;\frac&#123;a&#125;&#123;p&#125;\right&#41; &#61; a^&#123;\frac&#123;p-1&#125;&#123;2&#125;&#125;&#36; <p>We will provide an incomplete proof &#40;we don’t have time to cover all the steps&#41;.</p> <p><strong>Proof</strong>. First note that if &#36;a &#61; 0&#36;, then the statement is true by definition. Otherwise by Fermat’s Little Theorem, we have &#36;a^&#123;p-1&#125; &#61; 1&#36;. We write this as &#36;a^&#123;p-1&#125; - 1 &#61; 0&#36;. Factoring,</p> &#36;\left&#40;a^&#123;\frac&#123;p-1&#125;&#123;2&#125;&#125;-1\right&#41;\left&#40;a^&#123;\frac&#123;p-1&#125;&#123;2&#125;&#125;&#43;1\right&#41;&#61;0&#36; <p>Now if &#36;a&#36; is a quadratic residue, then &#36;a&#61;x^2&#36;. Then the first factor of this product becomes &#36;x^&#123;p-1&#125;-1&#36;, which by Fermat’s Little Theorem, is &#36;0&#36;. It is a theorem &#40;known as Lagrange’s Theorem&#41; that at most &#36;\frac&#123;p-1&#125;&#123;2&#125;&#36; values can make the first factor zero. Those must therefore correspond to the &#36;\frac&#123;p-1&#125;&#123;2&#125;&#36; nonzero quadratic residues. Otherwise, the second factor must be zero, which completes the proof of Euler’s Criterion.</p> <h2>Quadratic Reciprocity</h2> <p>The Law of Quadratic Reciprocity states that if &#36;p&#36; and &#36;q&#36; are odd prime numbers, then</p> &#36;\left&#40;\frac&#123;p&#125;&#123;q&#125;\right&#41;\left&#40;\frac&#123;q&#125;&#123;p&#125;\right&#41; &#61; &#123;&#40;-1&#41;&#125;^&#123;\frac&#123;p-1&#125;&#123;2&#125;\frac&#123;q-1&#125;&#123;2&#125;&#125;&#36; <p>We will not have time to prove this theorem. However, what it is saying, intuitively, is that unless &#36;p&#36; and &#36;q&#36; are both &#36;3&#36; modulo &#36;4&#36;, then &#36;p&#36; is a quadratic residue of &#36;q&#36; if and only if &#36;q&#36; is a quadratic residue of &#36;p&#36;. If both &#36;p&#36; and &#36;q&#36; are &#36;3&#36; modulo &#36;4&#36;, then we have a special case where the opposite occurs. Hence, this theorem is called the <strong>law of quadratic reciprocity</strong>.</p> <h3>Example</h3> <p>Show that there is no integer &#36;n&#36; such that &#36;2017 \mid n^2 - 5&#36; &#40;note that &#36;2017&#36; is an odd prime number&#41;.</p> <p><strong>Proof</strong>. We seek to show that &#36;5&#36; is not a quadratic residue of &#36;2017&#36;. By quadratic reciprocity, it suffices to show that &#36;2017&#36; is not a quadratic residue of &#36;5&#36;. Indeed, the quadratic residues modulo &#36;5&#36; are &#36;1&#36; and &#36;-1&#36;, and &#36;2017 &#61; 2 \not\in \&#123;-1, 1\&#125;&#36;.</p> </div>
